# task-2-raman
iris classification project 

## About the Project

This is a simple machine learning project built using Python and the K-Nearest Neighbors (KNN) algorithm. The program uses the Iris dataset to train a model and predict the type of flower based on its features.

The main goal of this project is to understand the basic steps of machine learning, such as loading data, training a model, making predictions, and checking the model's accuracy.

---

## Features

* Loads the Iris dataset
* Splits the data into training and testing sets
* Trains a KNN model
* Predicts flower classes
* Displays the prediction accuracy

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Install the required library (if needed):

```bash
pip install scikit-learn
```

4. Run the Python file:

```bash
python iris_classification.py
```
